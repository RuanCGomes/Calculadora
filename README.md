# ⚡ CdCEI (Calculadora de Consumo Elétrico Inteligente)

Este é o meu projeto da minha atividade como parte do meu curso de Desenvolvimento de Sistemas da EaDTEC.
O objetivo dele é calcular não só **quantos kiloWatts são gastos diariamente** mas também o seu **custo!**

Ele é um programa feito com o **Python**, usando a seguinte fórmula:

```
consumo = (potencia * tempo * 30) / 1000
custo = consumo * 0.75
```

Para usá-lo, basta apenas digitar o nome do aparelho, a sua potência em watts e a média do tempo que você o usa todo dia.
❗ **IMPORTANTE:** na hora de dar o valor da potência e do tempo, deve-se digitar **apenas números**, pois escrever um texto (string) vai quebrar o cálculo (que usará valor em float).

Este é o meu projeto da minha atividade como parte do meu curso de Desenvolvimento de Sistemas da EaDTEC.
O objetivo dele é calcular não só **quantos kiloWatts são gastos diariamente** mas também o seu **custo!**
Foi feito com um simulador de código **Python** on-line para a atividade de DSI da Agenda 05
